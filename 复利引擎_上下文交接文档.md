## 一、项目概述

**复利引擎 (Compound Engine)** 是一个 BTC 事件合约预测辅助工具，结合机器学习模型和 Web UI，提供实时行情、多模型预测、自动下单等功能。

- **品牌名**：复利引擎 Compound Engine
- **作者**：@OxMagic_
- **版本**：v1.0
- **风格**：机甲/赛博朋克，霓虹/扫描线效果，Orbitron 字体

## 五、重启后端命令（开发调试用）

```powershell
# 杀掉所有可能占端口的进程
Get-Process python -ErrorAction SilentlyContinue | Stop-Process -Force -ErrorAction SilentlyContinue
Get-Process -Name "复利引擎" -ErrorAction SilentlyContinue | Stop-Process -Force -ErrorAction SilentlyContinue
Start-Sleep -Seconds 2

# 启动新后端（必须用 Normal 窗口模式，Minimized会导致模型加载失败）
Start-Process "C:\Users\Lru1\AppData\Local\Programs\Python\Python312\python.exe" `
    -ArgumentList "compound_engine_server.py" `
    -WorkingDirectory "C:\Users\Lru1\Desktop\复利引擎_迁移包" `
    -WindowStyle Normal
```

**注意**：
1. exe 和 python 不能同时跑（共用7788端口）
2. 后端已加端口冲突自动清理，但建议手动杀进程更干净

## 七、关键配置项

### 后端端口
```python
PORT = 7788
```

### 模型特征
- 特征数量：111个
- 标签：±0.05%方向性（噪音样本丢弃）
- 趋势过滤：EMA21/55 + MACD + ADX

### 自动下单
- 币安规则：≥5u 整数
- 山寨所：≥3u 三位小数
- 默认赔率：0.8
